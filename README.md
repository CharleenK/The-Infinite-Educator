# The-Infinite-Educator

## Overview

The Infinite Educator was created to empower and support educators and childcare centers.

## Table of Contents

- [Usage](#usage)
- [Frontend Implementation](#frontend-implementation)

## Usage

You can perform the following actions:

- **Click on nav pages**: Click on a page to see contents os that page: Home, About, Services, Learning Events and Workshops.

- **Click on nav links**: Click the icon to be brought to the facebook account, send an email, or see the LinkedIn.

## Frontend Implementation

### Overview

The frontend of The Infinite Educator is built with **React** and **styled-components** for dynamic styling. The frontend is responsible for rendering the UI, handling user interactions.

### Key Components

- **Home Page (Home.js)**: Displays several images and introduces the concepts of The Infinite Educator.
- **About Page (About.js)**: Shows the Principles, Mission, and the creator.
- **Services Page (Servives.js)**: Gives brief description of the services offered.
- **Learning Events Page (LearningEvents.js)**: Breaks down into 3 catigories of specilization the skills available.
- **Workshops Page (Workshops.js)**: Details the specific experience of each workshop mentioned in the Learning Events page.

### Styling

- **Global Styles (GlobalStyles.js)**: Sets global styles for the application, ensuring a consistent look and feel across all components.
- **Styled Components**: Used extensively for individual component styling, ensuring modular and reusable styles.
- **Responsiveness**: The application is fully responsive and adapts to different screen sizes.
